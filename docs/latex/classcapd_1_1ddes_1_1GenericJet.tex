class capd\+::ddes\+::\+Generic\+Jet$<$ Time\+Point\+Spec, Data\+Spec, Vector\+Spec, Matrix\+Spec, is\+Interval $>$}

A class to represent a generic jet for any data type

We assume that Data\+Type has default constructor and is convertible form/to Vector\+Type, i.\+e. Data\+Type has\+:
\begin{DoxyItemize}
\item a constructor Data\+Type(\+Vector\+Type) or Data\+Type(const Vector\+Type\&)
\item a cast operator Vector\+Type()
\item a copy operator and copy constructor.
\item default constructor
\end{DoxyItemize}

Examples\+: capd\+::\+Vector and Sets types are compatible. ddes/storage/doubleton types are compatible.
