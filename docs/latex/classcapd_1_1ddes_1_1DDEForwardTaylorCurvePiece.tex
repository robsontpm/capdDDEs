\subsubsection*{template$<$typename Time\+Point\+Spec, typename Set\+Spec, bool is\+Interval = capd\+::\+Type\+Traits$<$typename Set\+Spec\+::\+Matrix\+Type\+::\+Scalar\+Type$>$\+::is\+Interval$>$\newline
class capd\+::ddes\+::\+D\+D\+E\+Forward\+Taylor\+Curve\+Piece$<$ Time\+Point\+Spec, Set\+Spec, is\+Interval $>$}

A class to represent a part of some curve \$x\$ over \$\mbox{[}t0, t1)\$ given by\+:

\$(J$^\wedge$\{\mbox{[}N\mbox{]}\}\+\_\+\{t0\}x)(t) + (n+1) $\ast$ \textbackslash{}int\+\_\+\{t0\}$^\wedge$\{t\} \textbackslash{}xi(s) (t0-\/s)$^\wedge$\{N\} ds\$

where \$\textbackslash{}xi(s) \textbackslash{}in R \textbackslash{}subset \textbackslash{}mathbb\{R\}$^\wedge$d\$

in the code below\+: d = dimension n = order

If the interval version is used, then the class can represent sets of Curves, so that \$J$^\wedge$\{\mbox{[}N\mbox{]}\}\+\_\+\{t0\} \textbackslash{}in x + A $\ast$ r0 + Q $\ast$ r\$, (x is here a middle point of the set and Jet is interpreted as a collection of coefficients). \$r0\$ is some vector of any given dimension, compatible with \$A\$. We denote the dimension of \$r0\$ as \$\+N0\$. \$Q\$ is a diagonal block matrix, so that \$\+Q\+\_\+ii\$ is orthonormal \$d \textbackslash{}times d\$ matrix and and \$Q\$ is 0 elsewhere. Vector \$r\$ is of dimension \$d $\ast$ n\$. we store matrices row-\/wise, ie in the code below m\+\_\+A\mbox{[}k\mbox{]} represents a \$d \textbackslash{}times N0\$ matrix and m\+\_\+Q\mbox{[}k\mbox{]} is \$\+Q\+\_\+kk\$ a \$d \textbackslash{}times d\$ matrix. Finally, we store \$r\$ and \$x\$ again component-\/wise, so that in code below m\+\_\+r\mbox{[}k\mbox{]} and \$m\+\_\+x\mbox{[}k\mbox{]}\$ is a \$d\$-\/dimensional vector. Therefore we have\+: \$(J$^\wedge$\{\mbox{[}N\mbox{]}\}\+\_\+\{t0\}x)\+\_\+\{\mbox{[}k\mbox{]}\} \textbackslash{}in m\+\_\+x\mbox{[}k\mbox{]} + m\+\_\+A\mbox{[}k\mbox{]} $\ast$ r0 + m\+\_\+Q\mbox{[}k\mbox{]} $\ast$ m\+\_\+r\mbox{[}k\mbox{]}\$

N\+O\+TE\+: we store \$m\+\_\+r0\$ as a one \$\+N0\$-\/dimensional vector, but for optimization purposes we hold the variable as a pointer -\/ to allow sharing of this vector among many instances (important in \mbox{\hyperlink{classcapd_1_1ddes_1_1DDESolutionCurve}{D\+D\+E\+Solution\+Curve}})

T\+O\+DO\+: (N\+OT U\+R\+G\+E\+NT) extract abstract interface that has eval, taylor, summa, etc? T\+O\+DO\+: (N\+OT U\+R\+G\+E\+NT) base interface is already done as \mbox{\hyperlink{classcapd_1_1ddes_1_1GenericJet}{Generic\+Jet}}, it has taylor (or eval).
