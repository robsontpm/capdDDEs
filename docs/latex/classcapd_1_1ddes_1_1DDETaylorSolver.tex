this function produces estimates that could be used to produce solution after time h = distance between grid points in the input curve. The solution will be valid on \mbox{[}t0, t0+h\mbox{]}, its enclosure will be as interval set in out\+\_\+\+Coeffs\+Enclosure up to order one higher than the out\+\_\+\+Phi\+\_\+coeffs\+\_\+t0 The var out\+\_\+\+Phi\+\_\+coeffs\+\_\+t0 will hold the Jet coefficients at point t0 computed for the middle point of the curve, out\+\_\+\+Jac\+Phi\+\_\+coeffs\+\_\+t0 will be the Jacobian of coeffs w.\+r.\+t. variables in out\+\_\+u, over the whole set of the curve. The value out\+\_\+\+Rem\+\_\+coeffs\+\_\+t0 is the remainder, but it will be 0 always in this implementation, as Jets are computed exactly for grid points. Then out\+\_\+\+Phi\+\_\+x\mbox{[}0\mbox{]} is the value at t = t0+h computed by the Taylor method, i.\+e. evaluation of out\+\_\+\+Phi\+\_\+coeffs\+\_\+t0(t0+h). out\+\_\+\+Jac\+Phi\+\_\+x\mbox{[}0\mbox{]} and out\+\_\+\+Rem\+\_\+x\mbox{[}0\mbox{]} are like in Taylor method for O\+D\+Es.

Curve.\+move() procedure then should be able to extract information needed to construct a succesive Curve\+Piece of the solution over \mbox{[}t0, t0+h) and to set current value by use of out\+\_\+\{Phi$\vert$\+Jac\+Phi$\vert$\+Rem\}\+\_\+x as it is customary in O\+D\+Es code.
