this function produces approximation to the solution after time h = distance between grid points in the input curve. The var out_Phi_coeffs_t0 will hold the Jet coefficients at point t0, then out_JacPhi_coeffs_t0 will be the Jacobian of coeffs w.r.t. variables in out_u. Then out_Phi_x[0] is the value at t = t0+h computed by the Taylor method, i.e. evaluation of out_Phi_coeffs_t0(t0+h). out_JacPhi_x[0] is like in Taylor method for ODEs.

It is designed to be used internally. Operators() and epsilonShift() use this.
